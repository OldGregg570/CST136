# CST136
C++OOP class assignments

# Build instructions
Alias "MinGW32-make" to "make":
>> doskey make=mingw32-make $*

To build all:
>> make

To build and run the test suite:
>> make test

Start the game:
>> make start

# Optional Commands

To run GCOV code coverage:

>> make cover
